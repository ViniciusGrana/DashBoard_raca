# DashBoard Raça: Análise de Dados Caninos com Python

O **DashBoard Raça** é uma ferramenta de análise exploratória de dados (EDA) desenvolvida em Python para visualização de informações sobre diferentes raças de cachorros, permitindo insights rápidos a partir de uma base de dados estruturada.

## 📊 Tecnologias e Visualização

O projeto utiliza bibliotecas poderosas do ecossistema de Ciência de Dados do Python:

| Ferramenta | Aplicação |
| :--- | :--- |
| **Pandas** | Manipulação e limpeza da base de dados (Excel) |
| **Plotly / Streamlit** | Geração de gráficos interativos e dashboard dinâmico |
| **Python 3** | Lógica de processamento de dados |

## 🛠️ Funcionalidades Principais

A ferramenta permite ao usuário interagir com os dados de forma visual e intuitiva:

*   **Importação Dinâmica:** Leitura de arquivos `.xlsx` para processamento de dados tabulares.
*   **Filtros de Raça:** Visualização específica por categorias e características.
*   **Gráficos Interativos:** Representação visual de tendências e distribuições na base de dados.

## 🎓 Contexto do Projeto

Desenvolvido como parte do módulo de **Análise de Dados** no Curso Técnico, este projeto demonstra competência em **Python para Data Science**, tratamento de dados com Pandas e criação de interfaces para visualização de informações estratégicas.
